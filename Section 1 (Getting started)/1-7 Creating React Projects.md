## Làm việc với dự án cục bộ

## Bước 1: Cài đặt dependencies

**Lệnh:**

bash

`npm install`

**Khi nào chạy:**

- Chạy lần đầu tiên khi tạo dự án[geeksforgeeks+2](https://www.geeksforgeeks.org/reactjs/how-to-setup-reactjs-with-vite/)​
    
- Cài đặt tất cả packages bổ sung được sử dụng bởi dự án[digitalocean+1](https://www.digitalocean.com/community/tutorials/how-to-set-up-a-react-project-with-vite)​
    
- Ví dụ: thư viện React package[digitalocean+1](https://www.digitalocean.com/community/tutorials/how-to-set-up-a-react-project-with-vite)​
    

## Bước 2: Khởi động development server

**Lệnh (với Vite):**

bash

`npm run dev`

**Chức năng:**

- Khởi động development server (máy chủ phát triển)[geeksforgeeks+2](https://www.geeksforgeeks.org/reactjs/how-to-setup-reactjs-with-vite/)​
    
- Cho phép truy cập và xem trước website đang làm việc[react+1](https://react.dev/learn/build-a-react-app-from-scratch)​
    
- Thường chạy tại `http://localhost:5173/`[logrocket+1](https://blog.logrocket.com/how-to-build-react-typescript-app-vite/)​
    

**Lưu ý quan trọng:**

- Nên **giữ server chạy** (keep that server up and running) miễn là đang làm việc với code[digitalocean+1](https://www.digitalocean.com/community/tutorials/how-to-set-up-a-react-project-with-vite)​
    
- Server process ngầm sẽ theo dõi (watch) các code files[react+1](https://react.dev/learn/build-a-react-app-from-scratch)​
    
- Tự động cập nhật preview website mỗi khi lưu thay đổi vào code[digitalocean+1](https://www.digitalocean.com/community/tutorials/how-to-set-up-a-react-project-with-vite)​
    
- Cung cấp cách làm việc rất hiệu quả: viết code và thấy những thay đổi ngay lập tức[react+1](https://react.dev/learn/build-a-react-app-from-scratch)​
